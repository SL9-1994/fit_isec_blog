「FIT ISec」とは

FIT ISecは福岡工業大学の学内サークル (現在は非公式) であり，主に情報セキュリティに関しての研究や実践，学習を行っています．
創設者兼リーダーは情報工学科の B1(2026 年現在)です．
私自身も，情報工学やサイバーセキュリティに関して初学者ですので，メンバー同士で高め合えるサークルにしたいと考えております！

サークル名の由来: Information Security の頭文字
